# 🍰 Sweet Treats Bakery Website

A responsive multi-page bakery website developed using **HTML, CSS, and JavaScript**. The project provides an interactive online bakery experience where users can browse products, manage a shopping cart, and place orders through a validated order form.

---

## ✨ Features

- 🏠 Responsive multi-page website
- 🍰 Browse cakes, cupcakes, donuts, and cookies
- 🛒 Shopping cart with Add to Cart functionality
- 🔢 Automatic quantity management
- ❌ Remove items from the cart
- 💰 Automatic subtotal and total price calculation
- 📋 Order form with client-side validation
- 💾 Local Storage for persistent cart data
- 📱 Sticky navigation bar
- 🎉 Order success page

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- DOM Manipulation
- Local Storage
